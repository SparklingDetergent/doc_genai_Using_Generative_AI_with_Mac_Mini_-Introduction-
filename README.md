### 生成AIにできること
- 例えば以下のような生成をすることが可能です。
  - 任意の生成AIチャットサービスをブラウザで開き、以下のプロンプトを入力し実行してみましょう。

<br/><br/>
#### コード生成の具体例：Powershellスクリプト作成
- **プロンプト:** Powershellでファイル又はフォルダの削除処理を実装してください。フェーズ１から順に完成度を高めるものとします。

<br/><br/>
#### 文章生成の具体例：生成AI導入資料作成
- **プロンプト:** 生成AIの導入資料を作成してください。フェーズ１から順に完成度を高めるものとします。

<br/><br/>
#### プロンプトエンジニアリングについて◇
https://github.com/SparklingDetergent/doc_genai_About_prompt_engineering/blob/main/README.md
- いったん回答が終わったら、「続けて」と入力し実行してみましょう。
  - そろそろ回答が出尽くしたと感じたら、「これまでの回答を総合して最高の結果を出してください」と入力して実行してみましょう。
<br/><br/>
- 回答内容が期待するものではない場合、会話の途中で具体的なプロンプトを入力するか、会話をリセットし最初に入力するプロンプトを具体化します。
  - 「共通関数にして」「コメントを加えて」「安全性を考慮して」「初学者にも分かりやすく」等と具体的に入力し実行してみましょう。

### 生成AIの実体
- 生成AIの実体は大量の学習データを機械学習により詰め込んだファイル、とイメージすると分かりやすいかもしれません。

```mermaid
graph LR
    A[大量のデータ] --> B[機械学習] --> |学習| C{生成モデル◇};
    A --> D[深層学習] --> |学習| C;
```

<br/><br/>
### 生成AIの働き
- 生成AIは入力されたプロンプトに対して学習データに近い内容でランダムに回答しているだけ、とイメージすると分かりやすいかもしれません。

```mermaid
graph LR
    G[プロンプト（入力）] --> E{生成モデル◇} --> |推論| F[新しいデータ<BR/>新しいコンテンツ];
```
